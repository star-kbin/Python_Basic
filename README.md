# Lesson 폴더 안내

이 폴더는 파이썬 기초 교육용 자료를 모아둔 디렉터리입니다. 주제별 학습용 노트북과 연습문제가 포함되어 있습니다.

## 주요 파일

- `python_01_자료형.ipynb`: 파이썬 숫자형, 문자열, 문자열 연산, 인덱싱, 슬라이싱 등의 기초 자료형 학습용 노트북
- `python_02_리스트.ipynb`: 리스트 생성, 인덱싱, 슬라이싱, 연산, 메서드 활용 등 리스트 자료형 학습용 노트북
- `python_03_튜플.ipynb`: 튜플 생성, 인덱싱, 슬라이싱, 연산 및 튜플의 불변성 관련 학습용 노트북
- `python_04_딕셔너리.ipynb`: 딕셔너리 생성, 키/값 접근, 추가, 삭제 등의 기초 딕셔너리 학습용 노트북
- `python_05_집합.ipynb`: 집합 생성, 중복 제거, 합집합/교집합/차집합 등 집합 연산 학습용 노트북
- `python_06_bool.ipynb`: 불 자료형, 비교 연산자, 논리 연산자, 조건식 이해를 위한 학습용 노트북
- `python_07_변수.ipynb`: 변수 선언, 할당, 값 교환, 기본 연산 등 변수 관련 학습용 노트북
- `python_08_if문.ipynb`: `if`, `elif`, `else` 조건문을 활용한 흐름 제어 학습용 노트북
- `python_09_while문.ipynb`: `while` 반복문 기초, 조건 반복, 누적 계산, `break`/`continue` 사용 예제 노트북
- `python_10_for문.ipynb`: `for` 반복문, `range()` 사용, 반복 처리 및 중첩 반복 학습용 노트북
- `python_11_함수.ipynb`: 함수 정의, 인자 전달, 반환값, 재사용성을 다루는 학습용 노트북
- `python_12_입출력.ipynb`: `input()`과 `print()` 기본 사용법, 형 변환을 포함한 입출력 학습용 노트북
- `python_13_파일읽고쓰기.ipynb`: 파일 읽기/쓰기, `with` 문 사용, 텍스트 파일 입출력 학습용 노트북
- `python_practice_50_questions.ipynb`: 기초 내용을 바탕으로 구성한 50문제 연습용 노트북
- `python_intermediate_20_questions.ipynb`: 기초 숙지를 전제로 한 중급 20문제 연습용 노트북

## 기타 파일

- `calculator.py`: **`python_12_입출력.ipynb`** 에서 `input()`, `print()`, 기본 연산을 활용한 간단한 계산기 실습 예제로 사용
- `coffee.py`: **`python_09_while문.ipynb`** 에서 `while` 반복문과 `break` 문을 활용한 커피 자판기 시뮬레이션 예제로 사용
- `새파일.txt`, `foo.txt`, `test.txt`, `한글파일.txt`: **`python_13_파일읽고쓰기.ipynb`** 에서 파일 읽기/쓰기 및 인코딩 처리 예제로 사용되는 테스트 파일
- `test01.ipynb`: 별도의 테스트용 노트북

## 사용 방법

1. 원하는 주제의 노트북 파일을 열어 학습하세요.
2. 각 셀을 실행하면서 코드 동작을 확인합니다.
3. 연습문제 노트북은 먼저 스스로 코드를 작성한 뒤, 정답 셀과 비교하세요.

## 주의 사항

- Jupyter Notebook 환경에서 실행할 때는 셀 순서를 지켜 실행하는 것이 좋습니다.
- `python_practice_50_questions.ipynb`와 `python_intermediate_20_questions.ipynb`는 파일 생성/읽기 문제를 포함하고 있으므로 실행 디렉터리를 확인하세요.
